The solution has been written in Scala Language.

The Attachement conatins the code base.

Download the zip file and extract it and compile it using sbt.

1. go inside the project folder and run the below command
	sbt package
2. The jar file requires two arguments to be passed to run.
	- hostname of the machine from which data needs to be read
	- output directory where the output files shoud be written
3. Run the producer and the this consumer simultaneously.
4. check the output folder.
